# **App Name**: Groq SalesAI

## Core Features:

- AI Campaign Generator: Generates marketing campaign strategies, content ideas, ad copy variations, and call-to-action suggestions, powered by Groq's LLaMA 3 70B. The campaign generator incorporates a 'tool' that decides whether or not to factor in the platform, target audience, or marketing objective.
- Intelligent Pitch Creation: Creates personalized sales pitches, including elevator pitches, value propositions, and key differentiators, leveraging Groq AI.
- Lead Qualification and Scoring: Analyzes lead attributes to generate a quantified lead score with detailed reasoning, probability of conversion, powered by Groq AI.
- Responsive UI: User-friendly and responsive HTML/CSS/JavaScript interface for accessing all platform features.
- API Integration: Connects the UI to the Flask backend and Groq API for seamless AI processing.
- Authentication: Secure user authentication with API key obtained from https://console.groq.com.

## Style Guidelines:

- Primary color: Deep blue (#2E3192), conveying trust and intelligence.
- Background color: Light blue (#D4DAED), providing a clean and modern feel. It has similar hue but very low saturation to the primary.
- Accent color: Purple (#845EC2), offering a vibrant contrast and highlighting key actions.
- Headline font: 'Space Grotesk', sans-serif, for a tech-focused, modern aesthetic.
- Body font: 'Inter', sans-serif, to ensure readability and a clean user experience.
- Code font: 'Source Code Pro' for any code snippets that may need to be shown.
- Modern and minimalist icons representing campaign types, sales stages, and lead qualities.
- Clean and structured layout with clear sections for campaign generation, pitch creation, and lead scoring.
- Subtle animations and transitions to enhance user experience and provide feedback during AI processing.
